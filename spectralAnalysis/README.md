### calspe.sh

Last update:  2014-05-15, 09:51 CST

#### Description:
This is a script package for spectra calculation from TDP outputfile 
of GENESIS

#### Usage:
```shell
calspe.sh outfile zentri_
    outfile: TDP output datafile
    zentri : z record order
```
